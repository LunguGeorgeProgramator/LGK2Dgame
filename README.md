# LGK2Dgame
Basic 2d game made in java 24, using a free tutorial.

# Packages

Java 24
gradle-8.14

# How to run

Run command 

````
./gradlew run
````

When the game window opens use keyboard w, s, d and a keys to move the white rectangle.
